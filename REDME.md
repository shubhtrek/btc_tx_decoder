# Bitcoin Transaction Decoder

A small Go tool that takes a raw Bitcoin transaction (in hex) and turns it into something easy to read.

--!! Still under construction !!--

### Why I'm building this
I want to really understand how Bitcoin transactions work under the hood, and it's great practice for writing clean, real-world Go code.

### What I want it to do
- Take raw transaction hex
- Show version, inputs, outputs, locktime, etc. in a clear way
- Help me learn both Bitcoin internals and good Go habits along the way
